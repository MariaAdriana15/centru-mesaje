Drag and Drop Multiple File Upload, Image Upload, and Google Drive Upload for WPForms.

== Description ==

Upload Fields for WPForms is an addon plugin for WPForms that allows you to add drag and drop multiple file upload, image upload, and Google Drive upload fields to your forms.

== VIDEO OVERVIEW ==
https://youtu.be/2WpLjqzXhUA

== FEATURES ==
**Drag and Drop Upload** - Upload files by dragging and dropping them into the upload field.

**Multiple File Upload** - Upload multiple files at once.
[Watch Video](https://www.youtube.com/watch?v=zweGYyzWSHA)

**Image Upload** - Upload images and display them in the form.
[Watch Video](https://www.youtube.com/watch?v=AP8PM7hMDhc)

**Google Drive Upload** - Upload files from Google Drive.
[Watch Video](https://www.youtube.com/watch?v=DIJvPYZyqRQ)

**File Size Limit** - Limit the size of files that can be uploaded.

**File Type Limit** - Limit the type of files that can be uploaded.

**File Count Limit** - Limit the number of files that can be uploaded.

**Save to Media Library** - Save uploaded files to the Media Library.

**Easy Google Drive Integration** - Easily integrate with Google Drive without having to create a Google API project.


== Frequently Asked Questions ==

= 1. How do I connect my Google Drive account? =
After activating the plugin you will get an option to connect your Google account in the `WPForms > Settings > Integrations` page.
Click the Google Drive Integration field and follow the instructions to connect your Google account.
